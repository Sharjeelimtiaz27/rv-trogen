║                                                                       ║
║   Privilege Escalation (Pattern 3):                                   ║
║   ┌─────────────────────────────────────────────────────────────────┐ ║
║   │ • WRITES privilege bits (active modification)                   │ ║

║   │ • Goal: System compromise                                       │ ║
